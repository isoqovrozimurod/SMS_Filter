# SMS_Filter
Excel ichidan **UZB telefon raqamlarni extract + normalize + deduplicate** qiluvchi oddiy GUI tool.

---

## ⚙️ Features
- UZB (`998`) raqamlarni aniqlash va filter  
- Noto‘g‘ri / foreign raqamlarni tashlab yuborish  
- Formatlash:
  - `+998 XX XXX XX XX`
  - `998XXXXXXXXX`
- Duplicate removal  
- Excel (`.xlsx`) ga export  
- Chunklab (masalan 700 tadan) sheetlarga bo‘lish  
- Custom prefix / suffix qo‘shish
